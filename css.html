What is CSS?

CSS (Cascading Style Sheets) is used to style and design web pages. While HTML creates the structure, CSS controls colors, layouts, fonts, spacing, animations, and responsiveness.

CSS Selectors

Selectors are used to target HTML elements.

p {
  color: white;
}

#box {
  background: blue;
}

.container {
  padding: 20px;
}

• Element Selector • Class Selector • ID Selector • Universal Selector • Group Selector

Colors & Backgrounds

body {
  background: linear-gradient(135deg, #000428, #004e92);
}

h1 {
  color: #ffcc00;
}

CSS supports HEX, RGB, RGBA, HSL, gradients, and background images.

The CSS Box Model

Every element has: Margin → Border → Padding → Content 

div {
  margin: 20px;
  padding: 15px;
  border: 2px solid white;
}

Flexbox Layout

.container {
  display: flex;
  justify-content: center;
  align-items: center;
}

Flexbox makes layout alignment easy and responsive.

CSS Grid

.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}

CSS Grid is powerful for creating advanced layouts.

Transitions

button {
  transition: 0.3s ease;
}

button:hover {
  transform: scale(1.1);
}

Animations

@keyframes slide {
  from { opacity: 0; }
  to { opacity: 1; }
}

.box {
  animation: slide 1s ease;
}

Responsive Design

@media (max-width: 600px) {
  body {
    background: black;
  }
}

Media queries allow websites to adapt to phones and tablets.

Advanced Topics

• CSS Variables • Pseudo Classes (:hover, :focus) • Pseudo Elements (::before, ::after) • Shadows • Filters • Transformations • Positioning (absolute, relative, fixed) • Z-index • Overflow • Object-fit • Scroll effects

Mini Project Example

.card {
  background: rgba(255,255,255,0.1);
  padding: 20px;
  border-radius: 15px;
  transition: 0.3s;
}

.card:hover {
  transform: translateY(-10px);
}

Try building a portfolio card using these styles!
