1. What is JavaScript?

JavaScript is a programming language used to create dynamic and interactive web applications.

2. Variables (let, const, var)

let age = 25;
const name = "Sam";
var city = "Paris";

3. Data Types

String, Number, Boolean, Undefined, Null, Object, Array, Symbol, BigInt

4. Operators

+, -, *, /, %, **  
== vs ===  
&&, ||, !

5. Conditions

if(age > 18){
  console.log("Adult");
} else {
  console.log("Minor");
}

6. Loops

 console.log(i);
}

7. Functions

function greet(name){
 return "Hello " + name;
}

map(), filter(), reduce(), forEach(), find()

 alert("Clicked!");

JSON.stringify()

// Supports multi-line strings

if (success) resolve("Done!");
else reject("Error!");
});

myPromise.then(res => console.log(res)).catch(err => console.log(err));

22. Async / Await

async function getData() {
try {
const response = await promise;
console.log(response);

console.error(err);
}
}

23. Fetch API

fetch('https://api.example.com/data')
.then(response => response.json())
.then(data => console.log(data));

24. Classes

class Animal {
constructor(name) {
this.name = name;
}
speak() {
console.log(${this.name} makes a noise.);
}
}

25. Modules

// export.js
export const add = (a, b) => a + b;

// import.js
import { add } from './export.js';

26. Closures

A function that remembers the environment in which it was created.

function outer() {
let count = 0;
return () => console.log(++count);
}
const counter = outer();
counter(); // 1

27. Hoisting

Variables (var) and function declarations are moved to the top of their scope during compilation.

console.log(x); // undefined
var x = 5;

28. Event Bubbling & Capturing

Bubbling: Event travels from child to parent. Capturing: Event travels from parent to child.

element.addEventListener("click", func, true); // true = Capturing

29. Callbacks

function process(callback) {
callback();
}
process(() => console.log("Callback executed"));

30. Higher Order Functions

Functions that take other functions as arguments or return them.

const multiplyBy = (num) => (x) => x * num;
const double = multiplyBy(2);

31. Recursion

function factorial(n) {
if (n === 1) return 1;
return n * factorial(n - 1);
}

32. Debounce & Throttle

Debounce: Executes after a delay of inactivity. Throttle: Executes at most once every X milliseconds.

33. Web APIs

Browser-provided tools: Geolocation, Web Storage, Canvas, History API.

34. REST APIs

HTTP Methods: GET (Read), POST (Create), PUT (Update), DELETE (Remove).

35. Event Loop

Manages the execution of multiple chunks of your script, handling the Call Stack and Task Queue.

36. Call Stack

LIFO (Last In, First Out) mechanism to keep track of function calls.

37. Memory Management

Allocation -> Use -> Release (Garbage Collection via Mark-and-Sweep algorithm).

38. OOP Principles

  • Encapsulation: Hiding data.
  • Inheritance: Sharing behaviors.
  • Polymorphism: Multiple forms.
  • Abstraction: Hiding complexity.

39. Functional Programming

Pure Functions, Immutability, and Avoiding Side Effects.
